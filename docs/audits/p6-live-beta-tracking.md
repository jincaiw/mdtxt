## Open P6 blockers

1. P6c: Windows Microsoft Pinyin and Linux IBus/Fcitx5 evidence. macOS Simplified Pinyin passed on `007843b`, but an unverified platform is not a pass for that platform.
2. P6d: Windows native WebView input and large-document evidence is still pending. Ubuntu native measurements passed; parser-only timing remains supporting evidence rather than a substitute for unverified native targets.
3. P6e: accessibility review, real macOS/Windows/Linux evidence and final requirement/rollback record.
