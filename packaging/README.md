# mdtxt packaging

mdtxt v0.1.0 is an internal preview build. No public package-manager manifest,
installer URL, or automatic update channel is published yet.

When a release channel is approved, create new manifests under this directory
for the mdtxt identifier and the approved release repository. Do not reuse the
upstream Paperling manifests or artifact URLs: their package identities, hashes,
publishers, and update lifecycle do not belong to mdtxt.
